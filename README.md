# Virtual Psychologist Frontend

Фронтенд для виртуального психолога с голосовым интерфейсом и AI-аватаром.

## Функции

- 🎙️ Голосовое общение (нажми и говори)
- 📝 Субтитры для ответов психолога
- 🔊 Озвучивание ответов (TTS)
- 🛡️ Определение кризисных ситуаций с предложением телефона доверия
- 🔄 Управление сессиями
